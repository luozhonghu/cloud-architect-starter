# 游戏行业云解决方案

## 1. 行业概述

### 1.1 游戏类型与特点

| 游戏类型 | 业务特点 | 技术挑战 | 典型案例 |
|----------|----------|----------|----------|
| **MMORPG** | 大规模多人在线，持久化世界 | 高并发、数据一致性、低延迟 | 魔兽世界、梦幻西游 |
| **MOBA** | 5v5对战，实时性要求极高 | 超低延迟、同步机制 | 英雄联盟、王者荣耀 |
| **FPS** | 快节奏射击，高精度判定 | 延迟敏感、防作弊 | CS:GO、绝地求生 |
| **卡牌游戏** | 回合制，策略性 | 防作弊、公平性 | 炉石传说、阴阳师 |
| **SLG** | 策略养成，多服务器 | 跨服同步、排行榜 | 文明、万国觉醒 |
| **休闲游戏** | 低门槛，碎片化时间 | 快速加载、广告变现 | 消消乐、地铁跑酷 |
| **超休闲游戏** | 极简玩法，病毒式传播 | 全球分发、低成本合成 | Flappy Bird |

### 1.2 核心业务需求

| 需求维度 | 说明 | 云服务支持 |
|----------|------|------------|
| **全球分发** | 低延迟接入，就近部署 | 全球CDN、多区域部署 |
| **弹性伸缩** | 应对活动期间流量激增 | 自动伸缩组、容器编排 |
| **实时通信** | 游戏内聊天、语音 | 即时通信、消息队列 |
| **防作弊** | 维护游戏公平性 | 行为分析、AI检测 |
| **数据安全** | 保护玩家账号和资产 | 加密存储、审计日志 |
| **快速迭代** | 频繁更新游戏内容 | CI/CD、灰度发布 |
| **数据分析** | 了解玩家行为，优化游戏 | 大数据分析、实时计算 |

**适用场景**：MMORPG、SLG等需要强一致性保证的游戏

**实现示例（Node.js）：**

```javascript
const WebSocket = require('ws');
const Redis = require('ioredis');

class GameServer {
    constructor() {
        this.wss = new WebSocket.Server({ port: 8080 });
        this.redis = new Redis();
        this.rooms = new Map();
        
        this.init();
    }
    
    init() {
        this.wss.on('connection', (ws) => {
            const playerId = this.generatePlayerId();
            
            // 玩家连接
            ws.on('message', async (message) => {
                const data = JSON.parse(message);
                
                switch (data.type) {
                    case 'login':
                        await this.handleLogin(ws, data);
                        break;
                    case 'join_room':
                        await this.handleJoinRoom(ws, data);
                        break;
                    case 'move':
                        await this.handleMove(ws, data);
                        break;
                    case 'attack':
                        await this.handleAttack(ws, data);
                        break;
                }
            });
            
            ws.on('close', () => {
                this.handleDisconnect(ws, playerId);
            });
        });
    }
    
    async handleLogin(ws, data) {
        // 验证玩家token
        const isValid = await this.verifyToken(data.token);
        if (!isValid) {
            ws.send(JSON.stringify({ type: 'error', code: 401 }));
            return;
        }
        
        // 获取玩家数据
        const player = await this.redis.hgetall(`player:${data.playerId}`);
        ws.playerId = data.playerId;
        ws.playerName = player.name;
        
        ws.send(JSON.stringify({
            type: 'login_success',
            player: {
                id: data.playerId,
                name: player.name,
                level: player.level,
                gold: player.gold
            }
        }));
    }
    
    async handleJoinRoom(ws, data) {
        const roomId = data.roomId;
        const playerId = ws.playerId;
        
        // 创建或加入房间
        let room = this.rooms.get(roomId);
        if (!room) {
            room = {
                id: roomId,
                players: new Map(),
                state: 'waiting'
            };
            this.rooms.set(roomId, room);
        }
        
        // 添加玩家到房间
        room.players.set(playerId, {
            id: playerId,
            name: ws.playerName,
            x: 0,
            y: 0,
            hp: 100,
            ws: ws
        });
        
        // 广播玩家加入
        this.broadcastToRoom(roomId, {
            type: 'player_joined',
            player: {
                id: playerId,
                name: ws.playerName,
                x: 0,
                y: 0
            }
        });
        
        // 发送房间状态给新玩家
        const roomPlayers = Array.from(room.players.values())
            .map(p => ({ id: p.id, name: p.name, x: p.x, y: p.y }));
        
        ws.send(JSON.stringify({
            type: 'room_joined',
            roomId: roomId,
            players: roomPlayers
        }));
    }
    
    async handleMove(ws, data) {
        const roomId = data.roomId;
        const room = this.rooms.get(roomId);
        
        if (!room) return;
        
        const player = room.players.get(ws.playerId);
        if (!player) return;
        
        // 更新玩家位置
        player.x = data.x;
        player.y = data.y;
        
        // 广播移动消息
        this.broadcastToRoom(roomId, {
            type: 'player_moved',
            playerId: ws.playerId,
            x: data.x,
            y: data.y
        }, ws);
    }
    
    async handleAttack(ws, data) {
        const roomId = data.roomId;
        const room = this.rooms.get(roomId);
        
        if (!room) return;
        
        const attacker = room.players.get(ws.playerId);
        const target = room.players.get(data.targetId);
        
        if (!attacker || !target) return;
        
        // 计算伤害（服务器端计算，防止作弊）
        const damage = this.calculateDamage(attacker, target);
        target.hp -= damage;
        
        // 广播攻击消息
        this.broadcastToRoom(roomId, {
            type: 'attack',
            attackerId: ws.playerId,
            targetId: data.targetId,
            damage: damage
        });
        
        // 检查是否死亡
        if (target.hp <= 0) {
            target.hp = 0;
            this.broadcastToRoom(roomId, {
                type: 'player_died',
                playerId: data.targetId
            });
        }
    }
    
    calculateDamage(attacker, target) {
        // 简化的伤害计算
        const baseDamage = 10;
        const randomFactor = Math.random() * 5;
        return Math.floor(baseDamage + randomFactor);
    }
    
    broadcastToRoom(roomId, message, excludeWs = null) {
        const room = this.rooms.get(roomId);
        if (!room) return;
        
        const messageStr = JSON.stringify(message);
        room.players.forEach((player, playerId) => {
            if (player.ws !== excludeWs && player.ws.readyState === WebSocket.OPEN) {
                player.ws.send(messageStr);
            }
        });
    }
    
    generatePlayerId() {
        return 'player_' + Math.random().toString(36).substr(2, 9);
    }
    
    async verifyToken(token) {
        // 实际应该调用认证服务验证
        return true;
    }
    
    handleDisconnect(ws, playerId) {
        // 从房间中移除玩家
        this.rooms.forEach((room, roomId) => {
            if (room.players.has(playerId)) {
                room.players.delete(playerId);
                this.broadcastToRoom(roomId, {
                    type: 'player_left',
                    playerId: playerId
                });
                
                // 如果房间为空，删除房间
                if (room.players.size === 0) {
                    this.rooms.delete(roomId);
                }
            }
        });
    }
}

new GameServer();
```

```python
import asyncio
import websockets
import json
from collections import defaultdict
import time

class FrameSyncServer:
    def __init__(self, fps=60):
        self.fps = fps
        self.frame_duration = 1.0 / fps
        self.rooms = defaultdict(dict)  # roomId -> {playerId: inputs}
        self.player_rooms = {}  # playerId -> roomId
        self.frame_count = 0
        self.running = False
        
    async def start(self):
        """启动帧同步服务器"""
        self.running = True
        
        # 启动帧同步循环
        asyncio.create_task(self.frame_sync_loop())
        
        # 启动WebSocket服务器
        server = await websockets.serve(self.handle_client, "0.0.0.0", 8080)
        print(f"帧同步服务器启动，FPS: {self.fps}")
        await server.wait_closed()
    
    async def frame_sync_loop(self):
        """帧同步主循环"""
        while self.running:
            frame_start = time.time()
            
            # 处理所有房间
            for room_id, room_inputs in self.rooms.items():
                if len(room_inputs) > 0:
                    # 广播帧数据
                    frame_data = {
                        'type': 'frame',
                        'frame': self.frame_count,
                        'inputs': list(room_inputs.values())
                    }
                    
                    await self.broadcast_to_room(room_id, frame_data)
                    
                    # 清空房间输入
                    self.rooms[room_id].clear()
            
            self.frame_count += 1
            
            # 保持帧率
            elapsed = time.time() - frame_start
            sleep_time = max(0, self.frame_duration - elapsed)
            await asyncio.sleep(sleep_time)
    
    async def handle_client(self, websocket, path):
        """处理客户端连接"""
        player_id = None
        room_id = None
        
        try:
            async for message in websocket:
                data = json.loads(message)
                
                if data['type'] == 'join':
                    # 加入房间
                    player_id = data['player_id']
                    room_id = data['room_id']
                    self.player_rooms[player_id] = room_id
                    
                    await websocket.send(json.dumps({
                        'type': 'joined',
                        'room_id': room_id,
                        'player_id': player_id
                    }))
                
                elif data['type'] == 'input':
                    # 收集输入
                    input_data = {
                        'player_id': data['player_id'],
                        'frame': self.frame_count,
                        'input': data['input']
                    }
                    
                    # 存储到房间
                    if room_id in self.rooms:
                        self.rooms[room_id][player_id] = input_data
        
        finally:
            # 客户端断开连接
            if player_id in self.player_rooms:
                del self.player_rooms[player_id]
                if room_id in self.rooms and player_id in self.rooms[room_id]:
                    del self.rooms[room_id][player_id]
```

#### 即时聊天系统

```java
@Service
public class GameChatService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Autowired
    private SimpMessagingTemplate messagingTemplate;
    
    /**
     * 发送聊天消息
     */
    public void sendChatMessage(ChatMessage message) {
        // 1. 敏感词过滤
        if (containsSensitiveWords(message.getContent())) {
            throw new BusinessException("消息包含敏感词");
        }
        
        // 2. 限流控制
        if (!checkRateLimit(message.getUserId(), message.getRoomId())) {
            throw new BusinessException("发送频率过快");
        }
        
        // 3. 保存聊天记录（异步）
        asyncSaveChatMessage(message);
        
        // 4. 广播消息到房间
        messagingTemplate.convertAndSend(
            "/topic/chat/" + message.getRoomId(), 
            message
        );
        
        // 5. 如果是私聊，发送给指定用户
        if (message.getType() == ChatMessage.Type.PRIVATE) {
            messagingTemplate.convertAndSend(
                "/queue/chat/" + message.getTargetUserId(), 
                message
            );
        }
    }
    
    /**
     * 获取房间聊天历史
     */
    public List<ChatMessage> getRoomChatHistory(String roomId, int limit) {
        String key = "chat:history:" + roomId;
        List<Object> messages = redisTemplate.opsForList().range(key, 0, limit - 1);
        
        return messages.stream()
            .map(msg -> (ChatMessage) msg)
            .collect(Collectors.toList());
    }
    
    /**
     * 敏感词过滤
     */
    private boolean containsSensitiveWords(String content) {
        String key = "sensitive:words";
        Set<String> sensitiveWords = redisTemplate.opsForSet().members(key);
        
        for (String word : sensitiveWords) {
            if (content.contains(word)) {
                return true;
            }
        }
        return false;
    }
    
    /**
     * 限流控制（Redis）
     */
    private boolean checkRateLimit(String userId, String roomId) {
        String key = String.format("chat:rate:%s:%s", roomId, userId);
        
        Long count = redisTemplate.opsForValue().increment(key);
        if (count == 1) {
            redisTemplate.expire(key, 60, TimeUnit.SECONDS);
        }
        
        // 每分钟最多10条消息
        return count <= 10;
    }
    
    /**
     * 异步保存聊天消息
     */
    @Async
    public void asyncSaveChatMessage(ChatMessage message) {
        // 保存到MongoDB
        chatMessageRepository.save(message);
        
        // 保存到Redis用于历史查询
        String key = "chat:history:" + message.getRoomId();
        redisTemplate.opsForList().leftPush(key, message);
        redisTemplate.opsForList().trim(key, 0, 99); // 保留最近100条
    }
}
```

### 3.3 排行榜系统

#### Redis排行榜实现

```python
import redis
from datetime import datetime, timedelta

class Leaderboard:
    def __init__(self):
        self.redis_client = redis.Redis(host='localhost', port=6379, db=0)
    
    def add_score(self, leaderboard_name, player_id, score):
        """
        添加或更新玩家分数
        
        Args:
            leaderboard_name: 排行榜名称
            player_id: 玩家ID
            score: 分数
        """
        key = f"leaderboard:{leaderboard_name}"
        self.redis_client.zadd(key, {player_id: score})
        
        # 设置过期时间（7天）
        self.redis_client.expire(key, 7 * 24 * 3600)
    
    def get_rank(self, leaderboard_name, player_id):
        """
        获取玩家排名
        
        Args:
            leaderboard_name: 排行榜名称
            player_id: 玩家ID
            
        Returns:
            排名（从1开始）
        """
        key = f"leaderboard:{leaderboard_name}"
        rank = self.redis_client.zrevrank(key, player_id)
        return rank + 1 if rank is not None else None
    
    def get_top_players(self, leaderboard_name, start=0, end=9):
        """
        获取排行榜Top N玩家
        
        Args:
            leaderboard_name: 排行榜名称
            start: 起始排名（从0开始）
            end: 结束排名
            
        Returns:
            玩家列表 [{player_id, score, rank}]
        """
        key = f"leaderboard:{leaderboard_name}"
        
        # 获取Top N
        result = self.redis_client.zrevrange(key, start, end, withscores=True)
        
        players = []
        for i, (player_id, score) in enumerate(result, start=start + 1):
            players.append({
                'rank': i,
                'player_id': player_id.decode('utf-8'),
                'score': int(score)
            })
        
        return players
    
    def get_players_around_me(self, leaderboard_name, player_id, count=5):
        """
        获取我周围的玩家（前后各count名）
        
        Args:
            leaderboard_name: 排行榜名称
            player_id: 玩家ID
            count: 前后各多少名
            
        Returns:
            玩家列表
        """
        key = f"leaderboard:{leaderboard_name}"
        
        # 获取玩家排名
        rank = self.redis_client.zrevrank(key, player_id)
        if rank is None:
            return []
        
        # 计算范围
        start = max(0, rank - count)
        end = rank + count
        
        return self.get_top_players(leaderboard_name, start, end)
    
    def increment_score(self, leaderboard_name, player_id, delta):
        """
        增加玩家分数（原子操作）
        
        Args:
            leaderboard_name: 排行榜名称
            player_id: 玩家ID
            delta: 分数增量
        """
        key = f"leaderboard:{leaderboard_name}"
        self.redis_client.zincrby(key, delta, player_id)
    
    def remove_player(self, leaderboard_name, player_id):
        """
        从排行榜移除玩家
        """
        key = f"leaderboard:{leaderboard_name}"
        self.redis_client.zrem(key, player_id)
    
    def get_player_score(self, leaderboard_name, player_id):
        """
        获取玩家分数
        """
        key = f"leaderboard:{leaderboard_name}"
        score = self.redis_client.zscore(key, player_id)
        return int(score) if score is not None else None
```

```java
@Service
public class AntiCheatService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    /**
     * 检测异常行为
     */
    public void checkAbnormalBehavior(String playerId, String action, 
                                       Map<String, Object> params) {
        // 1. 检测操作频率
        if (isAbnormalFrequency(playerId, action)) {
            flagSuspect(playerId, "abnormal_frequency", action);
            return;
        }
        
        // 2. 检测操作速度
        if (isAbnormalSpeed(playerId, action, params)) {
            flagSuspect(playerId, "abnormal_speed", action);
            return;
        }
        
        // 3. 检测异常数据
        if (isAbnormalData(playerId, action, params)) {
            flagSuspect(playerId, "abnormal_data", action);
            return;
        }
        
        // 4. 检测位置异常
        if (isAbnormalPosition(playerId, params)) {
            flagSuspect(playerId, "abnormal_position", action);
            return;
        }
    }
    
    /**
     * 检测操作频率异常
     */
    private boolean isAbnormalFrequency(String playerId, String action) {
        String key = String.format("anticheat:freq:%s:%s", action, playerId);
        
        Long count = redisTemplate.opsForValue().increment(key);
        if (count == 1) {
            redisTemplate.expire(key, 60, TimeUnit.SECONDS);
        }
        
        // 阈值：不同操作不同频率
        int threshold = getFrequencyThreshold(action);
        return count > threshold;
    }
    
    /**
     * 检测操作速度异常（瞬移）
     */
    private boolean isAbnormalSpeed(String playerId, String action, 
                                     Map<String, Object> params) {
        if (!action.equals("move")) {
            return false;
        }
        
        String key = String.format("anticheat:move:%s", playerId);
        
        // 获取上次位置和时间
        String lastMove = (String) redisTemplate.opsForValue().get(key);
        if (lastMove == null) {
            redisTemplate.opsForValue().set(key, 
                String.format("%d,%d,%d", 
                    params.get("x"), params.get("y"), System.currentTimeMillis()));
            return false;
        }
        
        String[] parts = lastMove.split(",");
        int lastX = Integer.parseInt(parts[0]);
        int lastY = Integer.parseInt(parts[1]);
        long lastTime = Long.parseLong(parts[2]);
        
        int currentX = (int) params.get("x");
        int currentY = (int) params.get("y");
        long currentTime = System.currentTimeMillis();
        
        // 计算距离
        double distance = Math.sqrt(
            Math.pow(currentX - lastX, 2) + 
            Math.pow(currentY - lastY, 2)
        );
        
        // 计算时间差（秒）
        double timeDiff = (currentTime - lastTime) / 1000.0;
        
        // 计算速度
        double speed = distance / timeDiff;
        
        // 更新位置
        redisTemplate.opsForValue().set(key, 
            String.format("%d,%d,%d", currentX, currentY, currentTime));
        
        // 速度阈值（单位/秒）
        double speedThreshold = 500.0;
        return speed > speedThreshold;
    }
    
    /**
     * 检测异常数据
     */
    private boolean isAbnormalData(String playerId, String action, 
                                     Map<String, Object> params) {
        // 检测伤害值异常
        if (action.equals("attack")) {
            int damage = (int) params.get("damage");
            int maxDamage = 1000; // 最大伤害值
            return damage > maxDamage;
        }
        
        // 检测金币异常
        if (action.equals("earn_gold")) {
            int gold = (int) params.get("gold");
            int maxGold = 10000; // 单次获得最大金币
            return gold > maxGold;
        }
        
        return false;
    }
    
    /**
     * 检测位置异常（边界检测）
     */
    private boolean isAbnormalPosition(String playerId, Map<String, Object> params) {
        if (!params.containsKey("x") || !params.containsKey("y")) {
            return false;
        }
        
        int x = (int) params.get("x");
        int y = (int) params.get("y");
        
        // 地图边界
        int minX = 0, maxX = 10000;
        int minY = 0, maxY = 10000;
        
        return x < minX || x > maxX || y < minY || y > maxY;
    }
    
    /**
     * 标记可疑玩家
     */
    private void flagSuspect(String playerId, String reason, String action) {
        // 记录到可疑玩家列表
        String key = "anticheat:suspects";
        redisTemplate.opsForSet().add(key, playerId);
        
        // 记录详细信息
        String detailKey = String.format("anticheat:detail:%s", playerId);
        Map<String, Object> detail = new HashMap<>();
        detail.put("player_id", playerId);
        detail.put("reason", reason);
        detail.put("action", action);
        detail.put("timestamp", System.currentTimeMillis());
        
        redisTemplate.opsForHash().putAll(detailKey, detail);
        
        // 如果可疑次数过多，自动封禁
        String scoreKey = String.format("anticheat:score:%s", playerId);
        Long score = redisTemplate.opsForValue().increment(scoreKey);
        if (score == 1) {
            redisTemplate.expire(scoreKey, 24, TimeUnit.HOURS);
        }
        
        // 超过5次可疑行为，封禁1小时
        if (score >= 5) {
            banPlayer(playerId, 3600, "自动封禁：可疑行为过多");
        }
    }
    
    /**
     * 封禁玩家
     */
    public void banPlayer(String playerId, int duration, String reason) {
        String key = String.format("anticheat:banned:%s", playerId);
        redisTemplate.opsForValue().set(key, reason);
        redisTemplate.expire(key, duration, TimeUnit.SECONDS);
        
        // 通知所有服务器
        notifyBan(playerId, duration, reason);
    }
    
    /**
     * 检查玩家是否被封禁
     */
    public boolean isBanned(String playerId) {
        String key = String.format("anticheat:banned:%s", playerId);
        return redisTemplate.hasKey(key);
    }
    
    /**
     * 获取不同操作的频率阈值
     */
    private int getFrequencyThreshold(String action) {
        switch (action) {
            case "move":
                return 30; // 30次/秒
            case "attack":
                return 10; // 10次/秒
            case "chat":
                return 5;  // 5次/秒
            default:
                return 20;
        }
    }
    
    private void notifyBan(String playerId, int duration, String reason) {
        // 通知所有游戏服务器
        // 实际实现可以使用消息队列
        System.out.printf("玩家 %s 被封禁 %d 秒，原因：%s%n", 
                         playerId, duration, reason);
    }
}
```

### 3.5 游戏热更新方案

```python
import requests
import hashlib
import json
import os

class GameUpdater:
    def __init__(self, update_url, game_path):
        self.update_url = update_url
        self.game_path = game_path
        self.version_file = os.path.join(game_path, 'version.json')
    
    def check_update(self, current_version):
        """
        检查是否有更新
        
        Args:
            current_version: 当前版本号
            
        Returns:
            (has_update, latest_version, update_size)
        """
        try:
            response = requests.get(
                f"{self.update_url}/check_update",
                params={'version': current_version}
            )
            data = response.json()
            
            if data['has_update']:
                return (
                    True, 
                    data['latest_version'], 
                    data['update_size']
                )
            else:
                return False, current_version, 0
        
        except Exception as e:
            print(f"检查更新失败: {e}")
            return False, current_version, 0
    
    def download_update(self, version, progress_callback=None):
        """
        下载更新包
        
        Args:
            version: 更新版本号
            progress_callback: 进度回调函数
            
        Returns:
            更新包文件路径
        """
        try:
            # 获取下载URL
            response = requests.get(
                f"{self.update_url}/get_update_url",
                params={'version': version}
            )
            data = response.json()
            download_url = data['download_url']
            file_hash = data['md5']
            
            # 下载文件
            update_file = os.path.join(self.game_path, 'update.zip')
            
            response = requests.get(download_url, stream=True)
            total_size = int(response.headers.get('content-length', 0))
            downloaded = 0
            
            with open(update_file, 'wb') as f:
                for chunk in response.iter_content(chunk_size=8192):
                    f.write(chunk)
                    downloaded += len(chunk)
                    
                    if progress_callback:
                        progress = (downloaded / total_size) * 100
                        progress_callback(progress)
            
            # 验证文件完整性
            self.verify_file(update_file, file_hash)
            
            return update_file
        
        except Exception as e:
            print(f"下载更新失败: {e}")
            return None
    
    def apply_update(self, update_file):
        """
        应用更新
        
        Args:
            update_file: 更新包文件路径
        """
        import zipfile
        
        try:
            # 备份当前版本
            backup_path = os.path.join(self.game_path, 'backup')
            self.backup_current_version(backup_path)
            
            # 解压更新包
            with zipfile.ZipFile(update_file, 'r') as zip_ref:
                zip_ref.extractall(self.game_path)
            
            # 更新版本号
            self.update_version()
            
            print("更新成功！")
        
        except Exception as e:
            print(f"应用更新失败: {e}")
            # 回滚
            self.rollback_backup(backup_path)
    
    def verify_file(self, file_path, expected_hash):
        """验证文件MD5"""
        md5_hash = hashlib.md5()
        with open(file_path, 'rb') as f:
            for chunk in iter(lambda: f.read(4096), b''):
                md5_hash.update(chunk)
        
        actual_hash = md5_hash.hexdigest()
        if actual_hash != expected_hash:
            raise Exception(f"文件校验失败: {actual_hash} != {expected_hash}")
    
    def backup_current_version(self, backup_path):
        """备份当前版本"""
        import shutil
        if os.path.exists(backup_path):
            shutil.rmtree(backup_path)
        shutil.copytree(self.game_path, backup_path)
    
    def rollback_backup(self, backup_path):
        """回滚到备份版本"""
        import shutil
        if os.path.exists(backup_path):
            # 清空当前目录
            for item in os.listdir(self.game_path):
                item_path = os.path.join(self.game_path, item)
                if os.path.isfile(item_path):
                    os.remove(item_path)
                elif os.path.isdir(item_path):
                    shutil.rmtree(item_path)
            
            # 恢复备份
            for item in os.listdir(backup_path):
                src = os.path.join(backup_path, item)
                dst = os.path.join(self.game_path, item)
                if os.path.isfile(src):
                    shutil.copy2(src, dst)
                elif os.path.isdir(src):
                    shutil.copytree(src, dst)
    
    def update_version(self):
        """更新版本号文件"""
        try:
            with open(self.version_file, 'r') as f:
                version_info = json.load(f)
            
            response = requests.get(f"{self.update_url}/latest_version")
            latest_version = response.json()['version']
            
            version_info['version'] = latest_version
            version_info['update_time'] = int(time.time())
            
            with open(self.version_file, 'w') as f:
                json.dump(version_info, f, indent=2)
        
        except Exception as e:
            print(f"更新版本号失败: {e}")


# 使用示例

        update_file = updater.download_update(latest_version, on_progress)

            updater.apply_update(update_file)
```
