# 核心服务 - 大模型服务

## 一、大模型服务概述

### 1.1 什么是大模型服务？

大模型（Large Language Model，LLM）服务是云厂商提供的基于大规模预训练模型的 AI 能力服务，包括：
- **对话模型**：通义千问、文心一言、智谱 AI 等
- **图像生成**：DALL-E、Midjourney、Stable Diffusion
- **语音合成**：TTS（文字转语音）
- **语音识别**：ASR（语音转文字）
- **多模态模型**：图文理解、视频生成

### 1.2 主流云厂商大模型服务

| 厂商 | 模型服务 | 主要模型 |
|------|---------|---------|
| **阿里云** | 通义千问 | Qwen-72B、Qwen-14B、Qwen-7B |
| **腾讯云** | 混元大模型 | Hunyuan-pro、Hunyuan-standard |
| **华为云** | 盘古大模型 | Pangu-3、Pangu-alpha |
| **百度云** | 文心一言 | ERNIE-4.0、ERNIE-3.5 |
| **字节跳动** | 云雀大模型 | Doubao-pro、Doubao-lite |

```python
def search_vector(query_vector):
    # 余弦相似度搜索
    sql = f"""
    SELECT id, content, 
           1 - (embedding <=> ARRAY{query_vector}) as similarity
    FROM documents

    LIMIT 10
    """
```

### 5.1 智能客服

```python
class IntelligentChatbot:
    def __init__(self):
        self.llm = self.init_llm()
        self.vector_db = self.init_vector_db()
    
    def chat(self, user_message):
        # 1. 检索相关知识
        context = self.search_knowledge(user_message)
        
        # 2. 构建提示词
        prompt = f"""
        你是一个智能客服助手，请根据以下知识库内容回答用户问题。
        
        知识库：
        {context}
        
        用户问题：
        {user_message}
        
        请给出准确、友好的回答：
        """
        
        # 3. 调用大模型
        response = self.llm.call(prompt)
        
        return response
    
    def search_knowledge(self, question):
        # 向量化问题
        vector = self.embed(question)
        
        # 检索相似文档
        results = self.vector_db.search(vector, top_k=5)
        
        return "\n".join([r['content'] for r in results])
```

#### 2. 缓存机制
```python
import hashlib
import redis

class CachedLLM:
    def __init__(self):
        self.redis = redis.Redis()
        self.llm = Generation()
    
    def call(self, prompt):
        # 生成缓存 key
        cache_key = hashlib.md5(prompt.encode()).hexdigest()
        
        # 检查缓存
        cached = self.redis.get(cache_key)
        if cached:
            return cached.decode('utf-8')
        
        # 调用 LLM
        response = self.llm.call(prompt)
        
        # 缓存结果（1小时）
        self.redis.setex(cache_key, 3600, response.output.text)
        
        return response.output.text
```

#### 3. 分级调用
```python
def smart_call(prompt):
    # 简单问题用小模型
    if len(prompt) < 100:
        return Generation.call(model='qwen-turbo', prompt=prompt)
    
    # 复杂问题用大模型
```

#### 4. 批量处理

```python
def batch_call(prompts):
    responses = []
    
    for i in range(0, len(prompts), 10):
        batch = prompts[i:i+10]
        
        # 并发调用
        with ThreadPoolExecutor(max_workers=10) as executor:
            futures = [executor.submit(call_qwen, p) for p in batch]
            responses.extend([f.result() for f in futures])
    
    return responses
```

```python
from dashscope import TextScan

def check_content_safety(text):
    response = TextScan.call(
        task='text_moderation',
        text=text,
        service='open_content_moderation'
    )
    
    if response.status_code == 200:
        result = response.output.results[0]
        
        if result.label != 'normal':
            return False, result.label
        
        return True, 'normal'
    else:
        return False, 'error'
```

#### 输入输出过滤
```python
class SafeLLM:
    def __init__(self):
        self.llm = Generation()
    
    def call(self, prompt):
        # 检查输入
        safe_input, label = check_content_safety(prompt)
        if not safe_input:
            return f"输入内容不合规：{label}"
        
        # 调用 LLM
        response = self.llm.call(prompt)
        
        # 检查输出
        safe_output, label = check_content_safety(response.output.text)
        if not safe_output:
            return "输出内容被过滤"
        
        return response.output.text
```

#### 核心代码
```python
class DocumentQA:
    def __init__(self):
        self.llm = Generation()
        self.embed_model = TextEmbedding()
        self.vector_db = AnalyticDBVector()
        self.oss = OssClient()
    
    def upload_document(self, file_path):
        # 1. 上传到 OSS
        oss_url = self.oss.upload(file_path)
        
        # 2. 解析文档
        text = self.parse_document(file_path)
        
        # 3. 文本切分
        chunks = self.split_text(text, chunk_size=500, overlap=50)
        
        # 4. 向量化
        for i, chunk in enumerate(chunks):
            vector = self.embed_model.embed(chunk)
            
            # 5. 存入向量数据库
            self.vector_db.insert({
                'id': f"{oss_url}_{i}",
                'content': chunk,
                'vector': vector,
                'metadata': {
                    'source': oss_url,
                    'chunk_index': i
                }
            })
        
        return {'success': True, 'document_id': oss_url}
    
    def ask(self, question, document_id=None):
        # 1. 向量化问题
        question_vector = self.embed_model.embed(question)
        
        # 2. 检索相关文档
        if document_id:
            filter_condition = f"metadata.source = '{document_id}'"
        else:
            filter_condition = None
        
        results = self.vector_db.search(
            vector=question_vector,
            top_k=5,
            filter_condition=filter_condition
        )
        
        # 3. 构建提示词
        context = "\n".join([r['content'] for r in results])
        
        prompt = f"""
        请根据以下文档内容回答用户问题。如果文档中没有相关信息，请如实告知。
        
        文档内容：
        {context}
        
        用户问题：
        {question}
        
        答案：
        """
        
        # 4. 调用大模型
        response = self.llm.call(
            model='qwen-plus',
            prompt=prompt,
            temperature=0.3
        )
        
        return {
            'answer': response.output.text,
            'sources': [r['metadata']['source'] for r in results]
        }
```

6. **持续优化**：监控性能、迭代模型、优化提示词

### 大模型 ≠ 万能

大模型服务能提供强大的 AI 能力，但需要：
- 合理的提示词工程
- 配合 RAG 架构
- 严格的成本控制
- 完善的安全机制
- 持续的优化迭代
